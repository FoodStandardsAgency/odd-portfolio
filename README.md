# ODD online portfolio

## Setup
Requirements:
Node.js 10+


## Install
```
npm install
```

## Run
```
npm start
```

## Access locally
http://localhost:3100

## .env file structure
```
NODE_ENV=development

PGHOST=''
PGUSER=''
PGDATABASE=''
PGPASSWORD=''
PGPORT=

COOKIE_SECRET = ''

CRON = 'n'
```
